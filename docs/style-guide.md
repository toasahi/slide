# レクタースライドスタイルガイド

## デザイン哲学

- **美麗でシンプル**: 情報過多を避け、視覚的にクリーン
- **色数制限**: グレースケール基調に限定的なアクセントカラー
- **可読性重視**: フォントサイズと余白を適切に
- **一貫性**: パターンの繰り返しで理解しやすく
- **アクセントカラー**: 1スライドあたり1-2色まで

---

## カラーパレット

### プライマリグラデーション

```css
background: linear-gradient(to right, #1B4565, #3E9BA4);
```

### グレースケール

| クラス | 用途 |
|--------|------|
| `gray-50` | パネル背景（淡い） |
| `gray-100` | パネル背景（やや濃い） |
| `gray-600` | サブテキスト |
| `gray-700` | 見出しテキスト |
| `gray-800` | メインテキスト |

### アクセントカラー

| 色 | コード | 用途 |
|----|--------|------|
| ダークブルー | `#1B4565` | タイトル、重要ラベル、グラデーション起点 |
| ティール | `#3E9BA4` | アクセント、グラデーション終点 |

---

## タイポグラフィ

| クラス | 用途 |
|--------|------|
| `text-em-3xl` | 数値強調、インパクトのあるキーワード |
| `text-em-2xl` | パネル見出し、セクションタイトル |
| `text-em-xl` | サブ見出し |
| `text-em-lg` | 本文、リスト項目 |
| `text-em-base` | 補足説明 |

---

## パネルデザイン

### 基本パネル

```html
<div class="bg-gray-50 rounded-xl shadow-lg p-6">
  内容
</div>
```

### 強調パネル（左ボーダー付き）

```html
<div class="bg-gray-50 rounded-xl shadow-lg p-6 border-l-4 border-gray-400">
  内容
</div>
```

### グラデーションパネル

```html
<div class="rounded-lg shadow p-4 text-center"
     style="background: linear-gradient(135deg, #1B4565 0%, #3E9BA4 100%);">
  <p class="text-white">内容</p>
</div>
```

## スライドタイトルと文体

### コロン（：）を使用しない

- ❌ 悪い例: `原則❶：自律性の最大化`
- ✅ 良い例: `原則❶ 自律性の最大化`

### 理知的な文体の維持

- **感嘆符（！）や疑問符（？）を避ける**
  - ❌ 悪い例: `AIで開発が10倍速に！`
  - ✅ 良い例: `AIで開発が10倍速に`

- **装飾的な絵文字を避ける**
  - ❌ 悪い例: `🧩 問題領域の複雑さ`
  - ✅ 良い例: `問題領域の複雑さ`

- **色による強調は控えめに**
  - `red-600`、`green-600` などは避け、`gray-600` / `gray-700` を使用

## 余白とスペーシング

### 推奨値

| 要素 | クラス |
|------|--------|
| グリッド間隔 | `gap-4` 〜 `gap-6` |
| マージン上 | `mt-4` 〜 `mt-6` |
| パディング | `p-4` 〜 `p-6` |

### オーバーフロー対策

- 縦5項目以上は2カラムに分割
- `gap-4` 以上で収まらない場合は `gap-2` に縮小
- `p-5` 以上で収まらない場合は `p-3` に縮小
